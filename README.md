# Text Utility App (ttu)

## Description

A lightweight Flask-based web application for text processing. This utility provides essential text transformation features including converting text to uppercase, lowercase, and reversing text. The application includes real-time character and word count statistics.

### Features
- Convert text to UPPERCASE
- Convert text to lowercase
- Reverse text
- Real-time word and character count
- Clean, responsive user interface
- Single-file deployment

## Installation

### Prerequisites
- Python 3.6 or higher
- pip (Python package installer)

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/mastar123-prog/some-repo.git
   cd some-repo
   ```

2. Create a virtual environment (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. Install Flask:
   ```bash
   pip install flask
   ```

4. Run the application:
   ```bash
   python ttu.py
   ```

5. Open your browser and navigate to:
   ```
   http://localhost:5000
   ```
